# Sidebar Menu

Sidebar Menu base `jQuery`

## Install

## Usage

import `css/js` files

```html
<link rel="stylesheet" href="./dist/sidebar-menu.css">
<script src="./dist/sidebar-menu.js"></script>
```

then invoke

```js
$.sidebarMenu(element)
```

## API

- `$.sidebarMenu(element)`
  - element element or jQuery element
